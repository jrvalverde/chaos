    3. huge (ca. 700 x 700 pixels)

PROGRAMMING LANGUAGE

The source files of the programs generating the images presented here heve been written in C language including only ANSI standard library headers.

ADVANTAGES

- DEVICE INDEPENDENCE - The source files can be compiled and run on any type of computer independently of the operating system and graphic environment.

- FAST EXECUTION - The compiled applications generate grayscale images onto graphic files in raw format. Therefore the graphic environment of the machine is not involved during execution and the application runs very fast (few minutes on a Spark 10 UNIX workstation; a little more on a PowerMacintosh 6100/66).

- EASY MANIPULATION OF IMAGE FILES - The graphic files can be suitably browsed and coloured by standard graphic manipulation applications like NCSA XImage and Photoshop or other.



IMAGE SIZES

- The SMALL (200 x 200) size allows you to examine the images sparing loading time.

- The LARGE (400 x 400) size is suitable to appreciate images in details and does not exceed usual monitors capability.

- The HUGE (700 x 700) size is advisable only if the connection is very fast and the monitor can display entirely a picture of such dimensions.
